# extended-spirit

モブプロ生配信ネタとして開発したが、公式でプロジェクトの複製ができるようになったためここに眠る

![](https://user-images.githubusercontent.com/20282867/152282131-dd9c8d9b-023e-4732-b1c9-4b4f22dfb0a2.png)

## prerequirements

- node
- yarn

## development

以下コマンドを実行

```
$ yarn
$ yarn build

# chrome://extensions を開き、生成された /dist 以下を読み込む
```
